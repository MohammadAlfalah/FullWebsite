FullWebsite – Web Application (Internship Project)
A full-stack web application developed during a software development internship at Virtual Data IT, Jordan (June – July 2025).

Overview
This project was built as part of a real internship assignment. It covers the full development cycle of a web-based application including backend logic, database integration, user management, and authentication.

Tech Stack
Backend: C# / ASP.NET Core (MVC)

Database: SQL Server / Entity Framework Core

Architecture: MVC (Model-View-Controller)

Version Control: Git

Features
CRUD operations for core data entities

User management and authentication logic

MVC structure with separated controllers, models, and views

Database integration with structured schema

Testing and issue analysis during development

Notes
This project was developed as part of a mandatory internship requirement (160 hours) and is kept private. Structure and logic reflect real-world web development workflows practiced during the internship.
